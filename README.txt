Práctica: Animaciones Web (CSS + JavaScript + GSAP)
====================================================

Estructura del proyecto
-----------------------
- index.html
- css/styles.css
- js/app.js

Cómo abrir
----------
1) Abre index.html en tu navegador.
2) Recorre el menú superior y prueba cada sección.

Qué demuestra cada sección
--------------------------
- Conceptos: definición de suavidad (easing), duración y retraso.
- Transiciones: botones con efectos de zoom, desplazamiento y cambio de color.
- Keyframes: tres cajas con aparecer, latido y flotar (animaciones continuas y de entrada).
- Micro‑interacciones: botón Like (squash) y Copiar enlace (toast sin librerías).
- Animaciones de Disney: tres demos (Anticipation, Squash & Stretch, Follow‑through) con botón para activarlas.
- JavaScript: reveal al hacer scroll con IntersectionObserver.
- GSAP: Intro y Secuencia de Títulos; Modal animado con rebote suave.
- Galería: 6 tarjetas con comportamientos distintos (zoom, rotate, color, lift, reveal, modal).
- Formulario: validación con “shake” al error y mensaje al éxito.
- Mapa: lista de dónde se cumple cada requisito para exponerlo claramente.
